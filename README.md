# Khám mắt Cộng đồng — GitHub Pages

## Cách triển khai

1. Tạo repository mới trên GitHub.
2. Tải toàn bộ nội dung **bên trong thư mục này** lên nhánh `main`.
3. Vào **Settings → Pages → Build and deployment**.
4. Tại **Source**, chọn **GitHub Actions**.
5. Mở tab **Actions** và chờ quy trình “Deploy GitHub Pages” hoàn tất.

Tên miền chính thức: https://khammatcongdong.matsaigontravinh.vn/

Gói đã có sẵn:

- Workflow GitHub Pages tự động chạy khi cập nhật nhánh `main`.
- Tệp `CNAME` cho tên miền riêng.
- SEO, Open Graph, Twitter Card, JSON-LD, robots và sitemap.
- Font Be Vietnam Pro hỗ trợ đầy đủ tiếng Việt.
- Giao diện responsive và các hiệu ứng chuyển động nhẹ.
